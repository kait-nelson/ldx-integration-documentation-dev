Introduction

Welcome to the developer documentation. Here you'll find everything you need to get started.

Example Code


// Initialize the API client
const client = new DevAPI({
    apiKey: 'your-api-key',
    environment: 'production'
});

// Make an API request
async function fetchUserData(userId) {
    try {
        const response = await client.users.get(userId);
        return response.data;
    } catch (error) {
        console.error('Error fetching user:', error);
        throw error;
    }
}
